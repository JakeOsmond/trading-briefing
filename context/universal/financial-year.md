# Financial Year

Holiday Extras operates on a **Financial Year To Date (FYTD)** running from **1 April to 31 March**. When anyone asks for "year to date", "FYTD", "this financial year", or "YTD" performance, use 1 April of the current financial year as the start date.

Named by the **year in which it ends**: 1 April 2026 – 31 March 2027 = **Financial Year 27 (FY27)**.

## Forecasting Cycle

| Forecast | Timing |
|----------|--------|
| **Budget (BGT)** | Set before FY starts. Fixed for the year. |
| **Forecast 1 (F1)** | Summer (few months in) |
| **Forecast 2 (F2)** | Autumn |
| **Forecast 3 (F3)** | Winter (if produced) |

Each forecast takes actuals for completed months + revised assumptions for remaining months. "Versus forecast" always means versus the **latest** forecast.
