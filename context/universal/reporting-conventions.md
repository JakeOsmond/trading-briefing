# Reporting Conventions and Abbreviations

Standard abbreviations used across all HX domains.

## Time Periods

| Abbreviation | Meaning |
|---|---|
| **TY** | This Year |
| **LY** | Last Year (Prior Year) |
| **WoW** | Week on Week |
| **YTD / FYTD** | Year to Date (Financial Year — 1 April to 31 March) |
| **MTD** | Month to Date |
| **LW** | Last Week |
| **BGT** | Budget (set once per FY, fixed) |
| **Fcast** | Forecast (updated in-year: F1 summer, F2 autumn, F3 winter) |

## Financial Metrics

| Abbreviation | Meaning |
|---|---|
| **GP** | Gross Profit |
| **GM** | Gross Margin (Revenue minus Supplier Costs) |
| **MAV** | Margin after VAT |
| **CpB** | Cost per Booking (PPC efficiency) |
| **Canx** | Cancellation |
| **Bkg / Bkgs** | Booking / Bookings |
| **Var** | Variance |
| **pp** | Percentage points |

## Business Units

| Abbreviation | Meaning |
|---|---|
| **HXC** | HX Cover (Insurance) |
| **HXA** | HX Adventures |
| **DACH** | Germany, Austria, Switzerland (European sub-region) |
| **R&I** | Retail & Intermediaries |

## Performance Comparisons

- **vs LY**: always compare against equivalent day-of-week (364-day offset), not calendar date
- **vs BGT**: Budget is set once before FY starts, remains fixed all year
- **vs Fcast**: "versus forecast" always means the latest forecast issued
